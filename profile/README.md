## イベントについて

ゲーム開発集会は、参加者みんなが一緒に作り上げる場です。

メインテーマは「**楽しさ紡ぐ霧幻庭園**」

「霧幻（むげん）」とは、霧のように広がり、どこまでも続く楽しさをイメージした言葉です。
この集会では、特定のリーダーが全体を仕切るわけではありません。
代わりに、参加者それぞれがテーマを理解し、自分のアイデアや創造性を持ち寄って、分散的に活動します。

まるで霧の中に浮かぶ不思議な庭園が、みんなのアイデアによって自然に形づくられていくようなイメージです。
各自が自分のペースで自由に楽しみながら、他の人の刺激を受けて作品がより豊かになっていきます。

「紡ぐ」という言葉には、一人ひとりが持ち寄る力が重なり合い、より優れた作品へと成長していく意味が込められています。
参加者みんなで、楽しさの連鎖が無限に続く「霧幻庭園」を紡ぎ出していく――
それこそが、このゲーム開発集会の目指す世界なのです。
